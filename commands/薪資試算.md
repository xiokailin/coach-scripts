# 薪資試算

**觸發：** `薪資試算`
**Tab：** `printf '\033]0;Salary-MM\007'`

---

## 使用前設定

請依你的合約填入以下數字，其餘流程不需修改。

| 設定項目 | 說明 | 填入你的值 |
|---------|------|-----------|
| 平均單堂課價 | 月收入 ÷ 月課數（建議每季更新） | NT$[填入] |
| 底薪 | 課堂抽潤低於底薪時的保底金額 | NT$[填入] |
| 勞健保扣除 | 每月固定扣除的勞健保金額 | NT$[填入] |

**抽潤率分級（依你的合約填入）：**

| 月課堂數 | 費率 |
|---------|------|
| 1 – [門檻1] 堂 | [費率1]% |
| [門檻1+1] – [門檻2] 堂 | [費率2]% |
| [門檻2+1] – [門檻3] 堂 | [費率3]% |
| [門檻3+1] 堂以上 | [費率4]% |

**業績達標獎金（依你的合約填入）：**

| 月業績金額 | 獎金 |
|---------|------|
| < [門檻1] 萬 | $0 |
| ≥ [門檻1] 萬 | +$[金額] |
| ≥ [門檻2] 萬 | +$[金額] |
| ≥ [門檻3] 萬 | +$[金額] |

---

## 執行流程

### 1. 詢問參數（一次詢問）

```
請提供以下資訊：

本月上課堂數（或範圍，如 148-170）：
本月業績總額（估算，用於判斷業績獎金）：
```

若使用者在課量預估後立即執行，可沿用當次的堂數區間，不需重問。

### 2. 計算公式

```
PT抽潤（稅前）= 堂數 × 平均單堂課價 × 抽潤率
薪資（稅前）  = PT抽潤 + 業績獎金
薪資（實領）  ≈ 薪資（稅前）− 勞健保扣除額
```

底薪保底：若 PT抽潤 < 底薪 → 實領改為「底薪 − 勞健保扣除額」

### 3. 輸出

```
【薪資試算 — YYYY/M】

| 堂數  | 費率   | PT抽潤     | +業績獎金  | 稅前合計    | 預估實領    |
|-------|--------|-----------|----------|------------|------------|
| N 堂  | XX.XX% | NT$XX,XXX | +NT$X,XXX| NT$XX,XXX  | NT$XX,XXX  |

業績達標獎金判斷：本月業績 NT$XXX,XXX → [達標/未達標] [對應獎金]
```

---

## 規則

- 若使用者提供堂數範圍（如 148–170），輸出每個費率門檻前後的關鍵節點
- 一對二課程計 1 堂（與課量估算一致）
- 業績達標獎金以月業績已成交金額（含應收未收）為準
- 平均單堂課價建議每季依實際薪資回推更新
