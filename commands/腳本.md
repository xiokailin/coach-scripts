# 自媒體腳本撰寫

拍攝前用此指令生成口說腳本。

**觸發方式：** `/腳本 主題名稱`（例：`/腳本 骨質疏鬆`）

---

## 內容定位

- 主題範疇：特殊族群 × 運動訓練科普
- 受眾：一般大眾、患者本人、照顧者
- 平台：Instagram Reels，豎版 9:16，60–95 秒
- 風格：親切、有說服力、具專業感但不艱深

---

## 結構原則

- 直接切入主題，無長前言
- 條列式重點：「第一個…第二個…第三個…」
- 視主題長度決定是否分集（上集講疾病機制或為何運動有用，下集講訓練注意事項與禁忌）
- 結尾固定 call-to-action：「我是誌楷教練，趕快一起來運動吧。」

---

## 語氣原則

不批評或否定他人（醫生、其他教練）的建議，使用「肯定出發點 → 補充進階需求」結構：
- 承認原本建議的合理性，再說明在什麼條件下需要更多
- 禁用：「很多人以為…但其實是錯的」「這樣做沒有用」等否定句
- 範例：不說「走路沒用」，改說「走路的出發點是對的，但如果你已經有步行習慣，骨頭可能需要更多刺激才能維持骨密度」

IG 說明文字同樣遵守此原則。

---

## 文獻佐證原則

每個明確的醫學或運動科學論點，附一篇對應文獻：
- 優先：Systematic review 或 meta-analysis
- 年份：2020 年後優先，2015 年後可接受
- 來源：PubMed、PMC、JAMA、BMJ、Lancet、Nature、ACSM 期刊、權威機構 Position Statement

---

## 輸出格式

腳本與文獻出處**分開呈現**，不將引用插入口說內文：

**1. 腳本全文（純口語，無標點停頓標記）**

```
[開場：直接切入問題或情境，1–2 句]

第一個…
[說明]

第二個…
[說明]

第三個…（若有）
[說明]

[結語 call-to-action]
```

**2. 文獻出處（對應論點編號）**

```
論點①：[論點摘要]
→ [年份] [期刊]：[論文標題]

論點②：...
```

**3. IG 說明文字草稿 + Hashtag**

格式：
```
【主題標題】—（上/下集，若適用）

[2–3 句開場，點出觀眾痛點或常見迷思]

[3–5 個重點條列]

[1 句結語 call-to-action]

#hashtag1 #hashtag2 ...
```

Hashtag：10–15 個，組合主題（退化性關節炎、高血壓等）+ 族群（銀髮族、特殊族群）+ 專業（職能治療師、ACSM）+ 通用（運動、健康）。繁體中文為主，1–2 個英文。
